# Лабораторная работа №2.2 (методы оптимизации). Задачи целочисленного программирования

1. Изучить и реализовать метод ветвей и границ. Python3. 
2. Продемонстрировать работу приложения на примерах. При демонстрации работы приложения выводить каждую итерацию метода Гомори (симплекс-таблицы) отдельно, также выводить решение (ответ), для метода ветвей и границ выводить дерево решений, то есть исходные данные по ветке и ответ, промежуточные таблицы можно не выводить. Предусмотреть все возможные варианты решения задачи, включая отсутствие решений, несколько решений и т.п.
3. Исследовать поведение метода при различных входных данных. 
Для метода ветвей и границ подобрать примеры, в которых количество веток зависит от порядка выбора переменной для ветвления. Подобрать несколько задач большой разрядности и измерить количество подзадач при различных стратегиях выбора веток для обхода. Сделать выводы.
Для метода Гомори подобрать несколько задач большой размерности и оценить, как увеличивается размерность матрицы при выполнении метода. Изучить влияние различных вариантов организации симплекс-таблицы (в памяти) при добавлении дополнительных строк и столбцов во время решения задачи. Сделать выводы.
